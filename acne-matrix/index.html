PCP First-Line Prescribing Rules: How Many Agents to Start?

Tier 1: Mild Comedonal
1 Agent (Monotherapy)
Start Topical Retinoid alone QHS (Adapalene 0.1% or Tretinoin 0.025%). Do NOT prescribe oral antibiotics or multi-topical cocktails.
Tier 2: Mild-Mod Inflammatory
2 Agents (Dual Therapy)
Start Topical Retinoid (QHS) + Benzoyl Peroxide (AM) OR fixed combination (Epiduo / Benzaclin). Maximum 2 active topicals!
Tier 3: Mod-Severe / Truncal
1 Oral + 2 Topicals
Start Oral Doxycycline 100 mg BID (max 3-4 mos) + Topical Retinoid + BPO. BPO is MANDATORY to prevent resistance.
Tier 4: Adult Female / Hormonal
Oral Hormonal + Topicals
Start Spironolactone 50-100 mg PO daily (or COC) + Topical Retinoid + BPO. Replaces need for chronic oral antibiotics!
Tier 5: Severe Nodular / Scarring
Oral Isotretinoin
Refer for Oral Isotretinoin Monotherapy (0.5-1.0 mg/kg/day to 120-150 mg/kg cumulative target via iPLEDGE). Discontinue all other acne drugs!
